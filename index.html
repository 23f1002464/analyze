Welcome to the Data Processing Pipeline

This project demonstrates a robust data processing workflow using Python, Pandas, and GitHub Actions.

It takes raw Excel data, processes it, and publishes the results automatically.

Explore the repository to understand the code, CI/CD setup, and how data transforms into actionable insights.

Our Workflow at a Glance

1. Data Ingestion

data.xlsx is converted to data.csv and committed to the repository.

2. Data Processing (Python)

execute.py reads data.csv, performs analysis, and outputs result.json.

3. CI/CD with GitHub Actions

Automated linting (Ruff), execution, and deployment of result.json to GitHub Pages.
